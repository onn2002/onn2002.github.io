Overview

The Crescendo College Parking System is a smart parking solution developed to improve and streamline the vehicle entry process at Crescendo International College. The primary aim of the project is to reduce the reliance on manual entry logging by introducing a semi-automated system that utilizes computer vision and microcontroller-based hardware to assist with vehicle recognition.

At the core of the system is the ESP32-CAM module, a low-cost microcontroller with an integrated camera, which is used to capture images of incoming vehicles. These images are triggered through a simple web interface, where the user can manually click a "Get Still" button to capture the current frame. Once captured, the image is automatically sent to a local processing server for further analysis.

The server runs a custom-trained YOLOv8 (You Only Look Once Version 8) model specifically tailored for number plate detection. This deep learning model is capable of accurately identifying and extracting the number plate region from the input image, even in varied lighting and environmental conditions. The use of YOLOv8 significantly improves detection accuracy compared to traditional image processing techniques.

Although the original goal was to implement a fully automated number plate recognition pipeline — including optical character recognition (OCR) using tools such as Tesseract — practical challenges such as inconsistent OCR accuracy and limited development time led to a change in approach. As a result, the system allows for manual entry of the number plate text after detection. This ensures accurate identification while still benefiting from automated number plate localization using the AI model.

Once the number plate text is confirmed, it is displayed on a P10 LED display board, which is commonly used for digital signage. This board is integrated with an Arduino controller that receives the number plate text and displays it in real time. The visual output serves both as confirmation for users and as a record of vehicle entry.

The Crescendo College Parking System highlights the effective combination of embedded hardware, artificial intelligence, and display technologies to address a real-world problem. Despite not achieving full automation, the system delivers a reliable, semi-automated solution that can be further expanded in the future. Potential improvements include refining OCR accuracy, enabling real-time processing, adding database logging for record-keeping, and implementing wireless control to further reduce manual steps.

This project demonstrates a strong foundation in hardware-software integration, practical application of AI in edge devices, and system design tailored to campus-scale deployment.
